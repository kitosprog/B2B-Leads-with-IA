# 📝 Pasos para Crear Repositorio en GitHub

## PASO 1: Abrir Página de Nuevo Repositorio

Abre esta URL en tu navegador:
👉 **https://github.com/new**

(Si no estás logueado, primero inicia sesión en GitHub)

---

## PASO 2: Llenar el Formulario

En la página que se abre, completa lo siguiente:

### Repository name (IMPORTANTE)
```
company-scraper-saas
```
(Escribe exactamente esto)

### Description (Opcional)
```
Professional B2B lead scraping SaaS with intelligent search
```

### Visibilidad
Elige UNA de estas opciones:
- 🔘 **Private** ← Recomendado (solo tú lo ves)
- ⚪ **Public** (todos pueden verlo)

### Initialize this repository with:
⚠️ **NO MARQUES NADA** (déjalo todo sin marcar):
- ❌ Add a README file
- ❌ Add .gitignore
- ❌ Choose a license

### Click en el botón verde:
```
[Create repository]
```

---

## PASO 3: Después de Crear

GitHub te mostrará una página con instrucciones.

En la sección **"…or push an existing repository from the command line"** verás algo así:

```
git remote add origin https://github.com/TU_USUARIO/company-scraper-saas.git
git branch -M main
git push -u origin main
```

**COPIA LA PRIMERA LÍNEA** (la que tiene `git remote add origin`)

Se verá así:
```
https://github.com/TU_USUARIO/company-scraper-saas.git
```

---

## PASO 4: Dime la URL

Pégame aquí esa URL que copiaste y yo haré todo el resto automáticamente.

---

## 🆘 ¿Problemas?

### No puedo ver la página
- Asegúrate de estar logueado en GitHub
- Abre manualmente: https://github.com
- Click en el "+" arriba a la derecha → "New repository"

### No tengo cuenta de GitHub
1. Ve a https://github.com/signup
2. Crea una cuenta (es gratis)
3. Verifica tu email
4. Vuelve a https://github.com/new
